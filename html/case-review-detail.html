初步诊断

1. 2型糖尿病

- 糖尿病视网膜病变Ⅰ-Ⅱ期

- 糖尿病周围神经病变

2. 高血压病2级（极高危）

3. 高脂血症

4. 超重（BMI 27.1kg/m²）

鉴别诊断

1. 1型糖尿病：多发生于青少年，起病急，"三多一少"症状明显，易发生酮症酸中毒，胰岛素绝对缺乏，需终身胰岛素治疗。该患者为中老年起病，起病隐匿，既往使用口服降糖药有效，可鉴别。

2. 其他特殊类型糖尿病：如线粒体糖尿病、MODY等，多有特殊病史或家族史，该患者无相关表现，暂不考虑。

3. 继发性高血压：如肾性高血压、内分泌性高血压等，该患者高血压病史长，无肾脏疾病、内分泌疾病证据，考虑为原发性高血压。

治疗方案

一、生活方式干预

1. 饮食控制：低盐、低脂、糖尿病饮食，每日总热量控制在1600-1800kcal，碳水化合物占50-55%，蛋白质占15-20%，脂肪占25-30%。

2. 运动治疗：每周至少150分钟中等强度有氧运动（如快走、游泳），每周至少2次力量训练，避免空腹运动。

3. 体重管理：目标BMI＜24kg/m²，争取3-6个月内体重下降5-10%。

4. 戒烟限酒：建议戒烟，限制饮酒量（男性每日酒精摄入量＜25g，女性＜15g）。

5. 血糖监测：每日监测空腹及餐后2小时血糖，每周至少监测1次夜间血糖。

二、药物治疗

1. 降糖治疗：

- 继续使用门冬胰岛素30，调整剂量为早22U、晚20U皮下注射，餐前15-30分钟注射。

- 加用二甲双胍肠溶片0.5g tid（餐中服用）。

- 加用达格列净10mg qd（晨服）。

2. 降压治疗：

- 继续服用缬沙坦氨氯地平片（80mg/5mg）qd，目标血压＜130/80mmHg。

3. 调脂治疗：

- 继续服用阿托伐他汀钙片20mg qn，目标LDL-C＜1.8mmol/L。

4. 并发症治疗：

- 糖尿病周围神经病变：甲钴胺片0.5mg tid，α-硫辛酸胶囊0.6g qd。

- 糖尿病视网膜病变：羟苯磺酸钙胶囊0.5g tid，每6个月复查眼底。

三、随访计划

1. 血糖监测：每日记录血糖，每周门诊复诊1次，根据血糖调整胰岛素剂量。

2. 实验室检查：每2周复查空腹血糖、餐后2小时血糖、尿常规；每3个月复查HbA1c、肝肾功能、血脂、电解质；每6个月复查眼底、神经传导速度、24小时尿蛋白定量。

3. 目标：HbA1c＜7.0%，空腹血糖4.4-7.0mmol/L，餐后2小时血糖＜10.0mmol/L。

健康指导

1. 糖尿病知识教育：了解糖尿病的病因、症状、并发症及治疗原则，提高自我管理能力。

2. 饮食指导：

- 主食定量，粗细搭配，多吃蔬菜，适量水果（选择低糖水果，如草莓、柚子等）。

- 低盐饮食，每日食盐摄入量＜5g，避免腌制食品。

- 低脂饮食，避免动物内脏、油炸食品，选择植物油（如橄榄油、花生油）。

- 适量蛋白质，选择瘦肉、鱼类、蛋类、豆制品等优质蛋白。

3. 运动指导：

- 选择适合自己的运动方式，如散步、慢跑、游泳、太极拳等。

- 运动时间：餐后1-2小时进行，每次30-45分钟，每周5-7次。

- 运动强度：以微微出汗、心率不超过（170-年龄）为宜。

- 运动注意事项：携带糖果，避免低血糖；穿着舒适的运动鞋和服装；避免在恶劣天气下运动。

4. 血糖监测指导：

- 正确使用血糖仪，定期校准。

- 记录血糖值、饮食、运动及药物使用情况。

- 出现低血糖症状（如心慌、出汗、手抖等）时，立即测量血糖，如＜3.9mmol/L，及时补充含糖食物（如糖果、饼干、果汁等）。

5. 足部护理：

- 每日用温水洗脚（水温＜40℃），保持足部清洁干燥。

- 选择舒适、透气的鞋袜，避免穿过紧的鞋子。

- 定期检查足部皮肤，如发现破损、红肿、水泡等异常，及时就医。

6. 定期复诊：严格按照医生要求定期复诊，不可自行调整药物剂量或停药。

7. 心理调适：保持良好的心态，避免紧张、焦虑等不良情绪，必要时寻求心理支持。
